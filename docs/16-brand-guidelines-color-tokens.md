# 16 - Brand Guidelines and Color Tokens

## 1. Brand Strategy

### 1.1 Brand Personality

`Apex Performance Club` is premium, disciplined, precise, energetic, and human. It is not loud for the sake of being loud. It feels like a club for people who take training seriously but want a welcoming, professional environment.

| Trait | Expression |
| --- | --- |
| Disciplined | Clean layouts, sharp hierarchy, consistent spacing, minimal noise. |
| Premium | High-contrast typography, rich neutrals, restrained accent color, strong photography. |
| Energetic | Crisp motion, decisive CTAs, athletic rhythm in section pacing. |
| Trustworthy | Transparent pricing, real trainers, visible facilities, clear policies. |
| Human | Warm copy, member stories, practical guidance, no empty hype. |

### 1.2 Brand Voice

| Voice Attribute | Use | Avoid |
| --- | --- | --- |
| Direct | "Choose your plan. Start this week." | Long inspirational filler. |
| Confident | "Built for strength, endurance, and consistency." | Aggressive intimidation. |
| Premium | "Coached programs, modern equipment, measurable progress." | Cheap discount-first messaging. |
| Supportive | "We will help you start at the right level." | Shame-based fitness language. |
| Specific | "Strength, conditioning, mobility, and recovery." | Generic "get fit now" copy. |

### 1.3 Design Philosophy

- Let real fitness spaces, trainers, and members carry the visual identity.
- Use contrast, alignment, and spacing instead of decoration.
- Make CTAs feel decisive and tactile.
- Keep dashboards practical and dense without feeling cramped.
- Keep curves restrained: 6px to 8px radius for cards and panels unless a control requires a pill shape.
- Use icons only where they improve scan speed.

### 1.4 Emotional Positioning

The visitor should feel:

- "This gym is serious and modern."
- "The staff will guide me, not leave me confused."
- "The facilities and trainers are credible."
- "Joining and managing my membership will be simple."
- "This is a premium experience worth paying for."

### 1.5 Audience Positioning

| Audience | Message |
| --- | --- |
| Beginners | Start with guided structure, clear coaching, and a plan matched to your level. |
| Working professionals | Efficient training, flexible class times, simple digital membership management. |
| Strength-focused members | Modern equipment, progressive training, experienced coaches. |
| Transformation seekers | Consistency, accountability, attendance visibility, and trainer-led plans. |
| Premium buyers | Clean facilities, polished service, transparent pricing, and professional operations. |

## 2. Color Strategy

The visual identity uses near-black graphite, clean porcelain, athletic volt green, precise electric cyan, and controlled coral. The palette avoids the generic black/red gym look while still feeling athletic.

Usage ratio:

- 55 percent neutral background and surfaces.
- 25 percent text and structural graphite.
- 10 percent volt green accent.
- 5 percent cyan support accent.
- 5 percent coral/warning/action accents.

## 8. Color Usage Rules

| Use Case | Rule |
| --- | --- |
| Primary CTA on public dark hero | Use `primary=#C8F24A` with `primary-foreground=#111214`. |
| Primary CTA on light pages | Use `primary=#111214` with white text; use volt as highlight or secondary CTA. |
| Links | Use cyan in dark theme and graphite/underlined treatment in light theme. |
| Critical warnings | Use coral/error only for real urgency, not decoration. |
| Dashboard charts | Use chart tokens consistently across dashboards. |
| Badges | Use semantic background + border + text colors; never rely on color alone. |
| Focus states | Use `ring` token with 2px visible outline and offset. |
